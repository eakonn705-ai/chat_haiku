# chat_haiku

軽量な静的ウェブプロジェクトです。リポジトリには単一の `index.html` を含み、ローカルで開いて動作確認できるシンプルなサンプルサイトを提供します。

**概要**
- **目的**: このプロジェクトは、静的なHTMLページを用いた小さなデモ・プロジェクトです。ブラウザで開くだけで動作します。
- **対象者**: フロントエンドの学習を始めた人、簡単な配布ページを作りたい人。

**特徴**
- 追加のビルドツールや依存関係が不要
- シンプルで読みやすい構成

**準備と実行**
1. ローカルでブラウザに `index.html` を直接ドラッグ＆ドロップして開く、または
2. 簡易HTTPサーバで配信して確認する（推奨）:

```bash
# Python 3 がインストールされている場合（プロジェクトルートで実行）
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

**使い方（基本）**
- ブラウザで `index.html` を開くとサイトが表示されます。
- ページ内容を編集したら、ブラウザをリロードして変更を確認してください。

**プロジェクト構成**
- `index.html` — メインの静的ページ
- `README.md` — このファイル
